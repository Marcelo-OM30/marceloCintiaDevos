# MarceloCintia Devos

> Aplicativo de devocionais cristãos evangélicos reformados — Leia, escreva e compartilhe sua fé.

---

## Sobre o Projeto

**MarceloCintia Devos** é um aplicativo mobile (iOS & Android) para leitura da Bíblia Sagrada, criação de devocionais pessoais e compartilhamento de cards/stories nas redes sociais.

Focado na tradição **evangélica reformada**, com versões da Bíblia Almeida e outras da mesma tradição.

---

## Funcionalidades

- **Bíblia completa** — versões ARC, ARA, ACF (João Ferreira de Almeida)
- **Pesquisa bíblica** — busca por palavra, frase ou referência
- **Devocionais** — escreva, salve e organize seus devocionais diários
- **Editor de Cards** — crie imagens com versículos para Instagram, WhatsApp e outras redes
- **Stories** — compartilhe sua fé com estilo
- **Autenticação** — cadastro e login seguro

---

## Stack

| Camada | Tecnologia |
|--------|-----------|
| Mobile | React Native + Expo |
| Navegação | Expo Router |
| Estilos | NativeWind (Tailwind) |
| Canvas/Editor | React Native Skia |
| Estado | Zustand |
| Dados | TanStack Query |
| Backend/Auth/DB | Supabase (PostgreSQL) |
| Storage | Supabase Storage |
